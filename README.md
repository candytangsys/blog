
# Tang Yu Portfolio Website 🐠

**小魚 | Tech & Leadership Portfolio**

A modern, responsive personal brand website showcasing technical skills, leadership experience, and SITCON Camp application. Built with React, TypeScript, Tailwind CSS, and Vite for blazing-fast performance.

## 🎯 Project Overview

Personal tech and leadership portfolio website for Tang Yu - showcasing:
- Technical skills (Java, Python, C#, C++, HTML, etc.)
- IT Support and productivity tool expertise
- Leadership and team collaboration experience
- SITCON Camp application and commitment to mentoring

## 🛠 Tech Stack

- **Frontend Framework**: React 18 with TypeScript
- **Build Tool**: Vite 5
- **Styling**: Tailwind CSS 3
- **Routing**: React Router 6
- **Animations**: Framer Motion 10
- **Fonts**: Noto Sans TC, Inter, Poppins

## 📚 Pages

1. **Home** - Hero section with stats and CTA buttons
2. **Skills** - Programming languages, IT support, productivity tools, and soft skills
3. **Experience** - Timeline view of leadership roles and work experience
4. **Projects** - Portfolio of learning projects and applications
5. **Why SITCON** - Mission statement, key traits, and commitment to mentoring
6. **Contact** - Contact form and social media links

## 🎨 Design System

- **Primary Color**: Deep Blue (#1e3a8a)
- **Secondary Color**: White (#ffffff)
- **Accent Color**: Cyan/Teal (#06b6d4)
- **Typography**: Noto Sans TC (Chinese), Inter/Poppins (English)
- **Theme**: Clean, modern, professional with warm personality

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```

Visit `http://localhost:5173` in your browser to see the project.

### Build for Production

```bash
npm run build
```

The optimized build will be in the `dist/` folder.

### Preview Production Build

```bash
npm run preview
```

## ✅ Customization

To customize this template for your own use:

1. Update personal information in component files
2. Replace the avatar placeholder in Home.tsx
3. Update project links and descriptions in Projects.tsx
4. Modify contact information in Footer.tsx and Contact.tsx
5. Customize colors in `tailwind.config.js`
